# Nuvio Badges

Minimalist black & white Nuvio filter config with resolution badges.

## Repo Structure

```
Badge/
├── nuvio-config.json
├── badges/
│   └── resolution/
│       ├── 4k.png
│       ├── fhd.png
│       ├── hd.png
│       └── 2k.png
└── README.md
```

## Groups

- **Quality** — 4K, 2K, FHD, HD
- **Releases** — Remux, BluRay, WEB-DL, WEBRip, SeaDex
- **Visual** — DV, HDR10+, HDR10, HDR, HLG, SDR, IMAX, 3D, AI (dark yellow fill)
- **Audio** — ATMOS, TRUEHD, DTS:X, DTS-HD MA, DTS-HD, DTS, DD+, DD
- **Codecs** — HEVC, H.264, AV1, XviD
- **Audio Channels** — 7.1, 6.1, 5.1
- **Language** — 14 flags + Multi
- **Special Tags** — SEADEX Release, IMAX
